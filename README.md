## Getting Started

### Prerequisites

```shell
nvm install lts
nvm use lts
npm i npm@latest -g
npm init -y
```

### Installation

```shell
$ npm i -D parcel
$ npm i -D handlebars
$ npm i -D sass

$ npm i -D express
```

## Useful commands

```shell
## Run in development mode
$ npm run dev

## Build project in folder 'dist'
$ npm run build

## Run in enterprise mode
$ npm run start

## Remove temporary folders
$ npm run clear
```
